The Supabase Data API automatically generates an API directly from your database schema. It exposes REST endpoints for your project's databases, including read replicas and load balancers. You can configure the API base URL, custom domains, and PostgREST settings for your project. The API URL provided is a RESTful endpoint for querying and managing your database. Enabling the Data API allows you to use any Supabase client library and PostgREST endpoints with any schema configured in the Settings tab.
